
Incident Response Playbook Generator
A web application designed to help Security Operations Centers (SOC) quickly generate MITRE-aligned incident response playbooks. This project was developed as part of an internship to automate the creation of tailored defense procedures based on specific threat scenarios.

Features
Dynamic Playbook Generation: Creates personalized response procedures using parameters like attack type, severity, industry, and detection source.

Indicator and Impact Tracking: Allows users to log observed indicators and assess business impact through an interactive interface.

Knowledge Exchange Community: Includes a collaborative section where analysts can share and review custom playbooks.

Export and Storage: Supports saving playbooks and downloading them as PDF documents for documentation and auditing.

Tech Stack
Frontend: React, HTML, CSS, Vite.

Backend: Python-based API engine.

Getting Started
Prerequisites
Ensure you have Node.js and Git installed on your system.

Installation
Clone the repository:

Bash
git clone https://github.com/kavyanpanchal2007-art/incidence_response_Generator.git
Navigate to the frontend directory:

Bash
cd playbook_frontend
Install dependencies:

Bash
npm install
Run the application:

Bash
npm run dev
